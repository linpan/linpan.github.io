pyhtontips:re.sub 高级用法

发表于 2018-03-23

场景

用正则实现：-3000 变成’-3,000’ and 45000 变成 ‘45,000’

re.sub(“^(-?\d+)(\d{3})”, ‘\g<1>,\g<2>‘, your_string)

注明：\g<1> 为命名的组

repl 参数分析

re.sub(pattern, repl, string)

repl，就是replacement，被替换，的字符串的意思。
repl可以是字符串，也可以是函数。
如果repl是字符串的话，其中的任何反斜杠转义字符，都会被处理的。
即：

  • \n：会被处理为对应的换行符；
  • \r：会被处理为回车符；
  • 其他不能识别的转移字符，则只是被识别为普通的字符：
    比如\j，会被处理为j这个字母本身；
  • 反斜杠加g以及中括号内一个名字，即：\g，对应着命了名的组，named group

当repl 参数是函数时

s_num = "hello 123 world 456";

def add_num_100(matched):
     s= matched.group("number");
     int_s = int(s);
     added_vaule = int_s + 111;
     return str(added_vaule);


replacedStr = re.sub("(?P<number>\d+)", add_num_100, s_num);

对于简单的逻辑无需单独起行写个函数，直接使用lambda函数实现。
