知识点：

  • 临时文件的自动创建及自动销毁 写入临时文件的是file-like 对象
    -ZipFile wirte方法的参数filename,要压缩的文件名
  • arcname 重新指定在压缩包的文件路径
    如果不指定arcname 压缩的文件包括文件的整个路径.我们可以通过arcname重新调整文件的压缩结构。
