django DRF 视图

发表于 2018-08-03

目前有4种方法创建api:

  • Pure Django views
  • APIView subclasses 自由程度比较高，适合定制
  • generics.* subclasses 封装一些
  • viewsets.ModelViewSet 高级封装

选择哪个方法, 基本原则是：

  • Use viewsets.ModelViewSet when you are goin to allow all or most of CRUD operations on a model.
  • Use generics.* when you only want to allow some operations on a model
  • Use APIView when you want to completely customize the behaviour.

翻译：

  • viewsets.ModelViewSet 适合对模型的大多数的CRUD操作
  • generics.* 适合一些操作
  • APIView 可以完全定义经典行为特性。

django普通视图 实现api

JsonResponse返回json
from django.shortcuts import render, get_object_or_404
from django.http import JsonResponse

from .models import Poll

def polls_list(request):
    MAX_OBJECTS = 20
    polls = Poll.objects.all()[:20]
    data = {"results": list(polls.values("question", "created_by__username", "pub_date"))}
    return JsonResponse(data)


def polls_detail(request, pk):
    poll = get_object_or_404(Poll, pk=pk)
    data = {"results": {
        "question": poll.question,
        "created_by": poll.created_by.username,
        "pub_date": poll.pub_date
    }}
    return JsonResponse(data)
