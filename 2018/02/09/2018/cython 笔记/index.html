cython文档笔记

发表于 2018-02-09 | 分类于 python

cythhon 语法

使用cdef 定义变量、结构体、常量

cdef int i, *j, k[100]

cdef struct node:
    int key
    float value

cdef enum:
    const = 1

或者block 一起：

cdef:
cdef struct node:
    int key
    float value
int i, *j, k[100]

或者使用ctypedef来定义类型名称

ctypedef double real
real num # double类型

函数

在Cython中有3种函数定义:

  • def: 传入python对象，返回python对象，直接调用
  • cdef: 传入python对象或C/C++值，返回python对象或C/C++值，不可直接调用, 可以使用指针

  • cpdef: 以上两者的混合,Python, Cython 都可以调用，不能使用指针

  • 类中的 cdef 定义C方法，而 cpdef 可以同时定义C方法和Python方法
    只有def和cpdef定义的函数在编译后可以在python代码中直接调用（run-time），cdef定义的函数则不能import 使用。 不过你可以使用def来对外封装

参数转换

如上所示，传递一个值是很简单的，只要稍稍注意一下它的类型。在python和C/C++之间有一些自动的类型转换:
struct 和python 字典类似；char* 代表 字符指针类

+-------------------------------------------------------------------------------------------+
|                C types                          |  From Python types  |  To Python types  |
| [unsigned] char [unsigned] short int, long      |  int, long          |    int            |
| unsigned int unsigned long [unsigned] long long |  int, long          |    long           |
| float, double, long double                      |  int, long, float   |    float          |
| char*   char* 代表 字符指针类                     |  str/bytes          |    str/bytes      |
| struct                                          |                     |    dict           |
+-------------------------------------------------------------------------------------------+

直接使用C/C++代码

如果只是一些C的函数需要封装进来
使用cdef extern可以把C代码中的函数声明到cython中:

cdef extern int add(int x, int y)

def add_py(int x, int y):
    return add(x, y)

类型声明

可以使用c的类型，也可以使用python 的类型

cdef list foo = []

cdef (double, int) bar # 返回类型可以使用元组

python参数转c参数类型：

def spam(python_i, python_s):
cdef int i = python_i
cdef char* s = python_s

return i, s


spam(2, b'abc') # b''字符串，不然报错

动态添加属性

有两种方式：

cdef class Animal:

cdef int number_of_legs
cdef dict __dict__

def __cinit__(self, int number_of_legs):
    self.number_of_legs = number_of_legs


dog = Animal(4)
dog.has_tail = True

# 属性存在dict里
>>> area.dog.__dict__
{'has_tail': True}

或者：

cdef class Animal:

cdef int number_of_legs

def __cinit__(self, int number_of_legs):
    self.number_of_legs = number_of_legs


class ExtendableAnimal(Animal):  # Note that we use class, not cdef class
    pass


dog = ExtendableAnimal(4)
dog.has_tail = True

cpython 导入

格式声明：

cimport module [, module...]

from module cimport name [as name] [, name [as name] ...]

模块定义在.pxd文件, 在pyx 导入

dishes.pxd:

cdef enum otherstuff:
    sausage, eggs, lettuce

cdef struct spamdish:
    int oz_of_spam
    otherstuff filler
restaurant.pyx:

from __future__ import print_function
cimport dishes
from dishes cimport spamdish

cdef void prepare(spamdish *d):
    d.oz_of_spam = 42
    d.filler = dishes.sausage

def serve():
    cdef spamdish d
    prepare(&d) #  取地址,因为形参是结构体指针, （* 取值） 传地址到 形参
    # 恶补c :array[20] &array是整个数组array的首地址，array是数组首元素的首地址（和&array[0]一样）
    print(f'{d.oz_of_spam} oz spam, filler no. {d.filler}')

当然还有其他的方式实现:

  • 共享c函数

volume.pxd:

cdef float cube(float)

volume.pyx:

cdef float cube(float x):
    return x * x * x

spammery.pyx:

from __future__ import print_function

from volume cimport cube

def menu(description, size):
    print(description, ":", cube(size),
          "cubic metres of spam")

menu("Entree", 1)
menu("Main course", 3)
menu("Dessert", 2)

  • 共享扩展类型

    shrubbing.pxd:

    cdef class Shrubbery:
    cdef int width
    cdef int length

    shrubbing.pyx:

    cdef class Shrubbery:
    def __cinit__(self, int w, int l):
        self.width = w
        self.length = l

def standard_shrubbery():
    return Shrubbery(3, 7)

    landscaping.pyx:

    cimport shrubbing
import shrubbing

def main():
    cdef shrubbing.Shrubbery sh
    sh = shrubbing.standard_shrubbery()
    print("Shrubbery size is", sh.width, 'x', sh.length)

    注意点： Shrubbing.standard_shrubbery() 访问需要 import shrubbing (standard_shrubbery 是python函数）

    扩展类型

    cython 允许以 python types，作为扩展类型，但是必须使用 cdef class 声明。

     cdef int width, height

     def __init__(self, w, h):
         self.width = w
         self.height = h

     def describe(self):
         print("This shrubbery is", self.width,
               "by", self.height, "cubits.")

其他Py_ssize_t

Py_ssize_t 是python C API的定义类型

Py_ssize_t allocated;  # 是数据类型（比int大），Py_ssize_t 有符号long类型, 长度参数的类型(整型或者 Py_ssize_t)在包含

Python.h 官方解释如下

对Python的C API进行广泛的更改，使用新的 Py_ssize_t类型定义而不是int，将允许解释器在64位平台上处理更多数据。此更改不会影响Python在32位平台上的容量
