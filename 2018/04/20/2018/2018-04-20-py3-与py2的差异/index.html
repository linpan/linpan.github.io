第一期: py3 与py2的差异

发表于 2018-04-20

apply()

2to3 fixer ☑ six support ☐

apply()函数用于当函数参数已经存在于一个元组或字典中时，间接地调用函数.
apply()内建函数自py2.3以后被移除，因为可以直接调用函数。

buffer()

2to3 fixer ☑ six support ☐

buffer() –> memoryview(),他们并不完全兼容，

下面代码在py2,py3都可以允许：

>>> import sys
>>> if sys.version_info > (3,): # 确定py的版本号
...     buffer = memoryview
>>> b = buffer('yay!'.encode())
>>> len(b)
4

callable()

2to3 fixer ☑ six support ☑

callable()在py3.0移除，然后在py3.2再次加进来，如果在python3.1使用不可以callable的对象需要抓起TypeError错误。
如果判断是否可调用函数，可以通过下列方法：

>>> def afunction():
...     pass

>>> any("__call__" in klass.__dict__ for
...     klass in type(afunction).__mro__)
True

>>> import collections
>>> isinstance(afunction, collections.Callable)
True

如果没有引用2to3兼容包的情况下，可以使用hasattr函数

>>> hasattr(bool, '__call__')
True

six模块定义了可调函数,为py3使用。

Classes

2to3 fixer ☐ six support ☐

在py2有两者类的类型 新样式和旧样式, 在py3剔除了旧样式。

Comparisions

2to3 fixer ☐ six support ☐

不要使用 cmp() 函数， __cmp__ 已经不再被支持。因此排序请使用 __lt__、 __eq__ 和 __hash__ 这些更有意义的比较。（现有的 cmp() 函数使用可以修改为 (a > b) - (a < b)，等价于 cmp(a, b) 。）

def cmp(a, b):
    return (a > b) - (a < b)

coerce() and __coerece__

2to3 fixer ☐ six support ☐

python3 正式被移除了

Dictionary 方法

2to3 fixer ☐ six support ☐

python2 字典有iterkeys() 、itervalue() 、iteritems() 方法, 返回类型是迭代器
python 3 变成了 keys values items 方法了。 ‘has_key’被移除。
如果做到py2 py3的兼容,使用try/except

>>> d = {'key1': 'value1',
...      'key2': 'value2',
...      'key3': 'value3',
... }

>>> try:
...     values = d.itervalues()
... except AttributeError:
...     values = d.values()

>>> isinstance(values, list)
False

>>> for value in values:
...     print(value)
value3
value2
value1

except

2to3 fixer ☐ six support ☐

python 2.6 之前的版本是

except (Exception1, Exception2), target:

python 2.6 - python 3.x 以更简洁的表达

except (Exception1, Exception2) as target:

Exception object

2to3 fixer ☐ six support ☐

在 py2 异常对象是可迭代对象和可索引的

>>> e = Exception('arg1', 'arg2')
>>> e[1]
'arg2'
>>> for a in e:
...   print a
...
arg1
arg2

在 py3 必须使用args属性, 此属性也可以在py2支持

>>> e = Exception('arg1', 'arg2')
>>> e.args[1]
'arg2'
>>> for a in e.args:
...   print a
...
arg1
arg2

exec

2to3 fixer ☐ six support ☐

py2,exec 是语句

>>> g_dict={}
>>> l_dict={}
>>> exec "v = 3" in g_dict, l_dict
>>> l_dict['v']
3

py3, exec 是函数,类似print与print()

>>> g_dict={}
>>> l_dict={}
>>> exec("v = 3", g_dict, l_dict)
>>> l_dict['v']
3

execfile

execfile() 函数可以用来执行一个文件。

2to3 fixer ☐ six support ☐

execfile 在py2 是语句, 在py3被抛弃,你可以通过exce来实现

exec(open(‘hello.py’).read())

此方法适合python任何版本。
