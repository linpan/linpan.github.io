argparse 小节笔记

发表于 2018-06-12

可选参数 和位置参数

可选参数 Optional argument
可选参数将由-前缀标识,例如 ‘-foo’ ‘–foo’

>>> parser.add_argument('-f', '--foo')

位置参数 Positional arguments
剩下的即为位置参数,且位置参数都是必须的(required)

>>> parser.add_argument('bar')

官方文档例子prog.py
import argparse

parser =argparse.ArgumentParser(description='Process some integers.')
parser.add_argument('integers', metavar='N', type=int, nargs='+',
                help='an integer for the accumulator')
parser.add_argument('--sum', dest='accumulate', action='store_const',
                const=sum, default=max,
                help='sum the integers (default: find the max)')

args = parser.parse_args()
print(args.accumulate(args.integers))

integers是位置参数,在执行的适合不 ‘integers’

--sum 是可选参数，

dest是目标参数的名称, dest允许设置自定义属性名字(属性别名)

const 接受参数 或者回调函数, 这是接受sum回调函数。

type 接受参数的类型(默认是string类型), type属性还可以接收一个回调函数，这个回调函数接收一个命令行参数并返回转换类型之后的值

defualt默认参数

choices 参数默认范围

nargs 参数个人 （+ ？*）
action action参数指定了参数是如何被处理的
help help=’…’，指定帮助提示信息。

>>> python prog 1 2 4 5
### Namespace(accumulate=<built-in function max>, t=[1243])
>>> python prog 1 2 4 5 --sum
### 10
